# AI Agent 开发：从原理到生产

面向具备 Python 基础的开发者，系统讲解 AI Agent 从原理到生产的完整链路。

## 五篇递进

| 篇章 | 内容 | 目标 |
|------|------|------|
| **第一篇 原理篇** | 本质、架构、生态 | 建立第一性原理层面的理解 |
| **第二篇 开发篇** | 从零实现到框架实战 | 掌握 Agent 开发核心技能 |
| **第三篇 增强篇** | RAG、规划、多 Agent | 让 Agent 从"能用"到"好用" |
| **第四篇 生产篇** | 评估、安全、部署、调试 | 从 Demo 到生产级系统 |
| **第五篇 前沿篇** | 商业与趋势 | 把握技术发展方向 |

## 快速开始

```bash
# 克隆代码仓库
git clone https://github.com/tniuli/ai-agent-book.git

# 进入项目目录
cd ai-agent-book

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 本书特色

- **第一性原理**：从根本原理出发讲解技术，先讲"为什么"，再讲"怎么做"
- **代码驱动**：关键代码片段不超过 30 行，所有代码可独立运行
- **实战导向**：每章包含可运行示例或案例分析
- **生产视角**：覆盖工程落地中的真实问题，非玩具 Demo

## 读者前置条件

**必须具备：**

- Python 基础：函数、类、装饰器、基本异步
- HTTP API 概念：请求/响应、JSON 格式
- 命令行操作：pip install、cd、python 运行

**不需要具备：**

- 机器学习背景（不会训练模型，只需调用 API）
- LangChain/LangGraph 经验（从零讲起）
- 前端开发经验（TypeScript 部分为可选补充）
